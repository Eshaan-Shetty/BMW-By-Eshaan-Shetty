# BMW Experience Website With Test Drive
A BMW, Responsive website built for my web programming lab project. The site is dedicated to the BMW M-Series, showcasing their Models, Performance Specs, Brand Heritage and Test Drive. 


## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## Files Included
- 'index.html' -> Home Page
- 'models.html' -> BMW M-Models 
- 'performance.html' -> M-Performance Specs
- 'heritage.html' -> Our Heritage
- 'gallery.html' -> BMW Photo Gallery
- 'faq.html' -> Frequently Asked Questions
- 'testDrive.html' -> Test Drive [Name and BMW Model]


## How to Run
There is no backend or local server setup required. 
1. Extract the project folder.
2. Download Visual Studio Code
3. Double-click `index.html` to open it in modern web browser.
